# Greeting App

A simple Node.js + Express demo that greets you by name.

## Setup & Run

1. Install dependencies:
```
npm install
```

2. Start the server:
```
node server.js
```

3. Open your browser and go to:
```
http://localhost:3001
```

## Push to GitHub

```bash
git init
git add .
git commit -m "Initial commit - greeting app"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/greeting-app.git
git push -u origin main
```
